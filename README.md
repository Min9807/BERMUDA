# BERMUDA
**팀 버뮤다 - 기억(remeBER)하고 듣자(MUsic) 일기(DiAry)를 통해**



# 💁서비스 소개

**BERMUDA**는 일기를 작성하면 감정분석을 통해 시그니처 음악을 제공해주는 앱 서비스입니다. 해당 일기는 작성한 위치 근방에 있는 사람들에게 공유하며 음악과 일기를 통한 공감을 형성할 수 있습니다.
